What is this?

This is a simple webapp that, using the Quill Library (and KaTeX), allows you to write text, and saves to local storage every ten seconds (trigger a manual save by pressing 'Save' in the toolbar.).

In beta; not much is included on top of standard quill yet.

Please submit bugs or outright broken things to ThisCatLikesCrypto/Website.
