Stronge is a minimal code editor written in Python

Features:

Write to any file type

if your code is in ESDlang you can execute it with our built in interpreter!

Better not make any errors, you can't change a previous line!

when you launch stronge, it will prompt you to come up with a filename, then a file extension, put your chosen file extension here or leave the input blank for .txt, then get to writing code! to clear a file, type filclr on a new line, to save a file and exit, type savex on a new line, and to run your ESDLang code, type savrun on a new line


looking for something new? how about Stronge 0.3? using the ins command, you can now edit previous lines in this version, and we added line numbers! This is however a beta product due to some janky code used for editing previous lines, if your code constantly refreshes every time you enter a line, it's because of how we had to manage editing the file

formatting the ins command: type ins, space and then the line number you would like to ammend, 2 colons will pop up and this is where you write your amended line
