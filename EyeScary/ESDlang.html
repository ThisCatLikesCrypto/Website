ESDlang - EyeScary Development

ESDlang (formerly weardlang, references still visible)

a simple programming language with a minimal syntax, currently in v2.0 closed testing

What would coding with it look like?

ESDlang is a programming language like very few, no brackets, quotes or tags are needed here! the first word you type is always a reserved word, anything after that is just requirements for a function which can be anything.

Here's how some basic code might look:


write hello
var
er
str
input what is your name?:
write hello, er , have a good day!

This code produces:
hello
what is your name?: (name)
hello, (name) , have a good day!


Features currently not working:

Variable names can only be 2 letters
Punctuation has to be separate from a variable for a variable to register
Random numbers can only go up to 9

when you extract the zip, edit the file named trun.txt, this is where you write your code

to create a variable make a line with var on, then give the variable a name on the next line, pick between num or str for your variable type and then give it a value (example values: radnom 2 3 , input input value here: , this is a string , 6)

to write a line of text, type write, then space and then write what you want to write, variable names are automatically converted to their values upon printing
